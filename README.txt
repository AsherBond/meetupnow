Meetup Now v0.1

REQUIRED TO RUN:
Apache Ant
Google AppEngine API

To run, make sure this directory is in the same directory as the google APPENGINE API folder, and use the command
ant runserver
to run a server on port 8080 on your computer. Then, to view the site, go to 'http://localhost:8080/' in your web browser.

To compile without running, use the command
ant compile
